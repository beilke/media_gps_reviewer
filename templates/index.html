How to use:
  1. Enter the full path to the directory containing your images
  2. Optionally check "Also show images that already have GPS" to include all images
  3. For proxy GPS suggestions, check "Find closest GPS metadata" and set time frame
  4. Click "Start Reviewing GPS" to begin
  5. Review each image and update coordinates as needed
  6. Click the "Show Heatmap" button to visualize the geographic distribution of your images
  7. The "Save All Changes" button will appear if proxy GPS values are assigned
