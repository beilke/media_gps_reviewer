How to Use
  1. Click "Browse Directories" to select a directory with photos
  2. On the directory page, you can set filtering options like showing images with existing GPS data
  3. Select a directory to begin the review process
  4. Review each image and update coordinates as needed
  5. Click the "Show Heatmap" button to visualize the geographic distribution of your images
  6. The "Save All Changes" button will appear if proxy GPS values are assigned
  1. Click "Browse CSV Files" to see available CSV files in the data/csv directory
  2. Select a CSV file to load and review the GPS data it contains
  3. Alternatively, upload a new CSV file with image paths and GPS information
  4. Review each image and update coordinates as needed
  5. The "Save All Changes" button allows you to save all updates at once
  6. Changes will be written back to the original CSV file and to the image files
