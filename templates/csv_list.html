CSV Format Guide

Your CSV file should contain the following columns:

Column Required Description
path Required Full path to the image file
datetime Optional DateTime when the photo was taken (ISO format)
latitude Optional Latitude coordinate (decimal)
longitude Optional Longitude coordinate (decimal)
gps_source Optional Source of GPS data (e.g., 'original', 'proxy', 'manual')
